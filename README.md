# 📌 7주차 과제[Mission7]

## 요구 사항

- 고양이 사진 API를 통해 사진과 폴더를 렌더링한다.
- 폴더를 클릭하면 내부 폴더의 사진과 폴더를 표시한다.
- 현재 경로를 표시한다.
- 루트 경로가 아닌 폴더는 첫 파일 자리에 뒤로가기 버튼을 표시한다.
- 사진을 클릭하면 고양이 사진을 모달 창으로 표시한다.
- 모달은 ESC 및 사진 바깥 영역을 클릭하면 닫힌다.
- API를 불러오는 동안에는 로딩 중임을 표시한다.
